# AgroHub

Projeto acadêmico da FIAP 2026 - Fase 3 PBL, Grupo 44.

## Proposta

Plataforma web que conecta pequenos produtores rurais a instituições sociais, facilitando a destinação de excedentes e alimentos fora do padrão comercial.

Alinhamento: ODS 2 - Fome Zero e Agricultura Sustentável, metas 2.1 e 2.3.

## Estrutura

- `index.html`: página inicial e seção Sobre, com âncoras internas.
- `cadastro.html`: Cadastro com abas para Produtor, ONG e Usuário.
- `produtor.html`: Painel do Produtor.
- `ong.html`: Painel da ONG.
- `contato.html`: Fale Conosco com formulário HTML5 válido.
- `css/agrohub.css`: design system e componentes `ah-*`.
- `js/main.js`: comportamentos compartilhados.
- `pitch.txt`: link do pitch e nomes dos integrantes.

## Como abrir

Abra `index.html` no navegador. O projeto é estático e usa Bootstrap via CDN.
